# 技术博客与研究报告平台

## 🚀 项目特色

- **现代化技术栈**: 基于Vite 7.x + Tailwind CSS 4.x构建
- **响应式设计**: 完美适配桌面端和移动端
- **动态内容管理**: 自动提取文章元数据，支持分类筛选和搜索
- **交互丰富**: 平滑动画效果、实时搜索、标签分类
- **高性能**: 优化的构建配置，快速的加载体验

## 🚀 快速开始

### 环境要求

- Node.js 18.x 或更高版本
- pnpm 8.x 或更高版本

### 安装依赖

```bash
pnpm install
```

### 开发服务器

```bash
pnpm dev
```

访问 http://localhost:5173 查看博客主页

### 构建生产版本

```bash
pnpm build
```

### 预览构建结果

```bash
pnpm preview
```

### 代码格式化

```bash
pnpm format
```

## 📖 使用指南

### 添加新文章

1. 在项目根目录创建新的HTML文件
2. 添加完整的meta标签：
   ```html
   <title>文章标题</title>
   <meta name="description" content="文章摘要内容">
   ```
3. 在`src/index.js`的`htmlFiles`数组中添加文件名
4. 系统会自动提取标题和摘要，生成文章卡片

### 样式开发

- 主要样式使用Tailwind CSS类
- 自定义样式在`index.html`的`<style>`标签中
- 响应式断点：移动端<768px，平板≥768px，桌面≥1024px

### 开发规范

- 项目使用中文界面，所有文本内容应使用中文
- 文章HTML文件应包含完整的meta description标签
- 避免修改DOM结构的核心类名和ID
- 新增文章时注意文件名编码，使用UTF-8格式
