操作指南

1. 初始状态检查

zpool status && zpool list

2. 添加新磁盘

zpool add poolname /dev/sdx

3. 执行一致性检查

zpool scrub poolname

4. 监控扫描进度

zpool status -T d poolname

核心命令快速参考

# 存储池管理
zpool create poolname /dev/sda
zpool add poolname /dev/sdb
zpool status
zpool list
# 数据集管理
zfs create poolname/data
zfs set compression=lz4 poolname
zfs snapshot poolname@backup
# 备份恢复
zfs send pool@snap | recv backup
zfs rollback pool@backup

最佳实践建议

定期执行 Scrub

每月至少执行一次数据完整性检查

启用压缩

使用 lz4 压缩算法提升存储效率

注意 RAID 0 风险

Stripe配置无冗余，重要数据需额外备份

性能优化配置

压缩设置
zfs set compression=lz4 poolname
禁用访问时间
zfs set atime=off poolname
记录大小
zfs set recordsize=128K poolname

故障排除与解决方案

设备故障

标记离线并替换：

zpool offline pool /dev/sdb
zpool replace pool /dev/sdb /dev/sdc

导入问题

强制导入存储池：

zpool import -f poolname
zpool import -d /dev/disk/by-id pool

清理标签

清除设备标识：

zpool labelclear /dev/sdd
wipefs -a /dev/sdd

  • • 谨慎启用去重功能
